# SYSU-Course
中大刷客

####基本功能
1. 登陆教务系统
2. 查询已选课程&可选课程
3. 选课&退课
4. 查看课程评论
5. 收藏课程
6. 搜索课程
7. 实时监听课程变化(**仅在Wifi下可使用**)

####进阶功能
1. 实时监课程并自动选课(**仅在Wifi下可使用**)
2. 匿名评论课程,并更新到服务器
